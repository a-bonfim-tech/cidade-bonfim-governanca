# Onboarding de Contribuidores (15 minutos)

Bem-vindo(a). Este repositório opera com governança leve, auditável e alinhada a DevSecOps. Siga os passos abaixo para contribuir com segurança e eficiência.

## Passo 1 — Visão Geral (3 min)
- Landing de Compliance (GitHub Pages):
  https://a-bonfim-tech.github.io/cidade-bonfim-governanca/
- Baseline ativo: gov-baseline-v1.0.0

## Passo 2 — Regras Essenciais (5 min)
Leia obrigatoriamente:
- Diretrizes Éticas:
  - PT: docs/inclusao-pt.md
  - EN: docs/ethical-inclusion-en.md
  - DE: docs/ethische-einbindung-de.md
- CHECKLIST.md
- Política de Exceções: docs/exceptions.md

## Passo 3 — Como Contribuir (5 min)
1. Crie um branch a partir do branch base protegido.
2. Faça commits claros (assinados quando aplicável).
3. Abra um Pull Request usando o template.
4. Aguarde revisão do CODEOWNER.

## Controles em Vigor
- PR obrigatório
- CODEOWNERS por escopo
- CI documental e de segurança
- Commits assinados (baseline)

## Evidências e Auditoria
- AUDIT.md (mapa canônico)
- Releases e tags de baseline
- Histórico de PRs e workflows

## Dúvidas ou Exceções
- Use o template de ISSUE de Governança
- Exceções devem ser justificadas e temporárias

Obrigado por contribuir com responsabilidade.
