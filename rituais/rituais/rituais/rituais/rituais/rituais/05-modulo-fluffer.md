# Módulo Fluffer – Função de Aquecimento

Fluffer, aqui, é uma função simbólica:

- preparar, aquecer, testar caminho;  
- mediar entre Backend e Frontend (entre bastidor e cena, entre mundos);  
- nunca ser produto final, sempre função de ponte.

Princípios:

- Fluffer não é descartável.  
- Fluffer não é lixo.  
- Fluffer tem contrato, dignidade, limite e proteção.

Tudo ligado a este módulo precisa respeitar Constituição, TPE, IAM e Dreieinheit.
