# KPIs de Governança e Qualidade

## Objetivo
Monitorar efetividade da governança, qualidade documental e disciplina DevSecOps.

## KPIs
- **Tempo médio de revisão (PR)**: ≤ 48h
- **Taxa de PRs com checklist completo**: 100%
- **Commits assinados (documentais)**: ≥ 95%
- **Falhas de CI por PR**: ≤ 10%
- **Exceções ativas**: 0 (ou justificadas e temporárias)

## Coleta
- GitHub PRs e Actions
- Tags de baseline e releases

## Revisão
- Mensal, com registro em release notes ou changelog.
