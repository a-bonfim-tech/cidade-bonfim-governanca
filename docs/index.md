# Governança Documental — Cidade Bonfim

Este espaço consolida a **governança documental** do repositório *cidade-bonfim-governanca*, alinhada a práticas **DevSecOps**, com foco em rastreabilidade, auditoria e responsabilidade humana no uso de tecnologia e IA.

## Baseline Ativo
- **gov-baseline-v1.0.0** — Governança documental congelada
  - Diretrizes éticas (PT, EN, DE)
  - Checklist de conformidade
  - Política de exceções
  - Branch protegido, CODEOWNERS e CI

## Documentos-Chave
- **Diretrizes de Inclusão Ética**
  - Português (PT): `docs/inclusao-pt.md`
  - English (EN): `docs/ethical-inclusion-en.md`
  - Deutsch (DE): `docs/ethische-einbindung-de.md`
- **Checklist de Conformidade:** `CHECKLIST.md`
- **Política de Exceções:** `docs/exceptions.md`
- **Compliance (âncora pública):** `COMPLIANCE.md`
- **Mapa de Evidências:** `AUDIT.md`
- **Contribuição:** `CONTRIBUTING.md`

## Controles em Vigor
- Commits assinados (GPG)
- Pull Requests obrigatórios
- CODEOWNERS por escopo
- CI documental e de segurança
- Baselines versionados por tag

## Auditoria
Auditorias devem referenciar:
- Tags de baseline (ex.: `gov-baseline-v1.0.0`)
- Releases associadas
- Histórico de Pull Requests e workflows

## Transparência
Este repositório adota governança proporcional: **controle suficiente para auditar, leve o bastante para não travar o fluxo**.
