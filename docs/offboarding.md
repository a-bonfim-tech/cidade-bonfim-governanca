# Offboarding de Contribuições

Este checklist aplica-se ao encerramento de contribuições relevantes.

## Checklist
- [ ] PRs associados foram mergeados ou fechados
- [ ] Exceções (se houver) foram encerradas ou regularizadas
- [ ] Evidências estão registradas (PR, tag, release)
- [ ] Documentação atualizada, se aplicável

## Encerramento
Após este checklist, a contribuição é considerada concluída e auditável.
