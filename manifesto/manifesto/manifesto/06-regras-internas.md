# Regras Internas da Governança Bonfim

Regras não negociáveis:

1. Nada que destrua o Eu Bonfim é aprovado, mesmo que pareça sedutor no curto prazo.  
2. Nenhum pacto simbólico é válido sem log escrito neste repositório.  
3. Toda queda vira dado para patch diário, não munição para auto-tortura.  
4. Tecnologia não manda no sistema. Se vira vício, é reconfigurada.  
5. Rituais não substituem tratamento real: terapia, remédios e acompanhamento médico permanecem obrigatórios.

Este arquivo é o firewall da Cidade Bonfim.
