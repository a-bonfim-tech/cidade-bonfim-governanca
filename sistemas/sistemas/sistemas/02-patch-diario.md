# Patch Diário – Atualização Interna Contínua

Rotina enxuta:

1. Logar o dia (o que aconteceu de relevante).  
2. Anotar vulnerabilidades percebidas (emocionais, técnicas, relacionais).  
3. Definir um hotfix pequeno para o dia seguinte.  
4. Registrar um aprendizado.

Patch diário é ferramenta de governança, não mecanismo de punição.
