# Identidade Coletivo 3 (3C)

Os três Coletivos Donos são a instância superior de decisão da Cidade Bonfim.

Cada Coletivo representa:

- um eixo de valor,  
- um tipo de responsabilidade,  
- um tipo de cuidado.

Este documento descreve como o 3C:

- revisa decisões,  
- impõe limites,  
- corrige rota,  
sempre com foco em preservação do Eu Bonfim.
