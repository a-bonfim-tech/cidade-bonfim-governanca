# Governança Documental — Visão Executiva

Este repositório implementa uma governança documental alinhada a DevSecOps, com baselines versionados, commits assinados, PRs auditáveis e enforcement proporcional.

## O que está em vigor
- Diretrizes éticas (PT, EN, DE)
- Checklist e política de exceções
- Branch protegido e CODEOWNERS
- CI de documentação e segurança
- Baseline congelado (gov-baseline-v1.0.0)

## Evidência
- Tags e releases
- Histórico de PRs
- AUDIT.md como mapa canônico
