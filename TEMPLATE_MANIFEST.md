# DevSecOps Governance Template — Manifesto

Este repositório pode ser utilizado como template para implementação de governança DevSecOps **somente se** os controles aqui descritos forem mantidos.

## Controles Obrigatórios
- Commits assinados (GPG)
- Pull Requests obrigatórios
- CODEOWNERS por escopo
- Branches protegidos
- CI documental e de segurança
- Baselines versionados por tag
- Evidência pública (AUDIT.md)

## O que este template fornece
- Estrutura de governança documental
- Fluxo de contribuição auditável
- Políticas de exceção e offboarding
- Relatórios automáticos de governança
- Landing pública de compliance

## O que este template não permite
- Remoção silenciosa de controles
- Uso sem rastreabilidade
- Alterações fora de Pull Request
- Declarações não lastreadas em evidência

## Uso Responsável
A adoção deste template implica aceitação explícita das práticas de governança aqui definidas.
