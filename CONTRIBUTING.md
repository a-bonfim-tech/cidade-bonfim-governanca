# Contribuindo com este Repositório

Obrigado pelo interesse em contribuir. Este repositório adota práticas de governança documental alinhadas a DevSecOps.

## Princípios
- Responsabilidade humana no uso de IA
- Rastreabilidade e auditabilidade
- Mínimo privilégio e clareza de escopo

## Antes de Contribuir
- Leia a Diretriz de Inclusão Ética de Conteúdo (PT, EN, DE)
- Consulte o CHECKLIST.md
- Verifique a Política de Exceções

## Processo
1. Crie um branch a partir do branch base protegido
2. Faça commits claros (assinados quando aplicável)
3. Abra um Pull Request usando o template
4. Aguarde revisão e aprovação

## Auditoria
Contribuições podem ser auditadas a qualquer tempo com base nas tags de baseline.
